# TheAdvue - Premium Digital Agency Website

## Overview
A motion-driven, data-heavy digital agency website built with React, Motion (Framer Motion), and Tailwind CSS v4.

## Key Features
- **Scroll-triggered animations** with custom easing (cubic-bezier(0.22, 1, 0.36, 1))
- **Line-by-line text reveals** for all headings and content
- **Count-up animations** for statistics and metrics
- **Video placeholders** ready for real content replacement
- **23 Instagram Reels** linked and ready to showcase
- **Complete Meta Ads performance data** from 7+ real case studies
- **Google Business Profile & Local SEO data** from 5+ industries
- **Dark theme** with cyan/blue/teal accent system
- **60fps performance target** with GPU-accelerated transforms

## Animation Components

### AnimatedText
- Line-by-line text reveal
- Customizable delay
- Supports all heading levels

### AnimatedContainer
- Container animation with directional options
- Supports up/down/left/right entry
- Viewport-triggered

### CounterAnimation
- Number count-up effect
- Easing: ease-out-cubic
- One-time animation on scroll into view

### PerformanceTable
- Row-by-row staggered animation
- Hover effects
- Responsive design

### VideoPlaceholder
- Professional placeholder for video content
- Play icon and descriptive labels
- Hover effects with glow

- Intersection Observer for scroll triggers
- Lazy animation loading
- Viewport margin optimization (-50px)
- Passive scroll listeners
